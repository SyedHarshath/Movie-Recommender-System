**Movie Recommender System** | *Python, Scikit-Learn, NLP, Streamlit, TMDB API*

• Designed and implemented a content-based recommendation engine using natural language processing and vector-space similarity techniques on the TMDB 5000 movie dataset.

• Performed text preprocessing, feature engineering, and vectorization using CountVectorizer, and generated personalized recommendations using Cosine Similarity over 4,800+ movie records.

• Built and deployed a full-stack Streamlit application with TMDB API integration for dynamic poster retrieval, gaining hands-on experience in ML model deployment, cloud hosting, API integration, and production debugging.
